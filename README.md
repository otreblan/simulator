# Ufdtd

## Dependencies

* `asciidoctor`
* `bash-completion`
* `cmake`

## Build
``` bash
git clone https://github.com/otreblan/ufdtd
cd ufdtd

mkdir -p build
cd build

cmake ..
make
```
